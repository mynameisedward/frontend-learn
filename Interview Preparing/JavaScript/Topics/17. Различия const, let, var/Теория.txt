------------------ let & const ------------------

let и const появились в ES6 (ECMAScript 2015)

Нельзя обращаться до объявления

let и const имеют блочную область видимости



------------------ var ------------------

Переменные объявленные с var имеют функциональную область видимости
    Они доступны только в пределах текущей функции или глобального объекта, если функции нет

Переменные объявленные через var всплывают
    т.е можно вызвать до объявления, и значением будет undefined



------------------ Отличие var, let const ------------------

1) var всплывает

2) разная область видимости ( let, const ) vs ( var )

    let, const имеет блочную область видимости
        т.е нельзя вызвать за пределом блока
        можно вызывать только на вложенных уровнях

    var имеет функциональную область видимости
        т.е переменную можно использовать за пределом блока
        но не функции

3) const это константа
