-------------------- Promise.all() --------------------

Принимает коллекцию промисов, (чаще всего массив), 
    и возвращает новый промис, который будет выполнен, когда будут выполнены все промисы,
        или отклонен, если хотя бы один промис завершится с ошибкой

Метод возвращает массив значений всех переданных промисов,
    при этом сохраняя порядок оригинального массива, но не порядок выполнения

Если хотя бы один промис будет отклонен, то остальные проигнорируются 
    обработчик then() будет проигнорирован, и будет выполнятся код из catch()




-------------------- Promise.allSettled() --------------------

Похож на Promise.all(),
    возвращает новый промис, который выполнится когда выполнятся все переданные промисы
        полученный промис содержит массив результатов выполнения всех промисов, 
        сохраняя порядок оригинального массива



-------------------- Promise.any() --------------------

Дожидается выполнения всего одного промиса

Принимает массив промисов, 
    Возвращает новый промис, который выполнится, 
        когда выполнится первый из промисов из массива
    Возвращает значение первого успешно выполненного промиса

Промис может быть отклонен, 
    если ни один из промисов не выполнится

Если все промисы завершились с ошибкой, 
    метод Promise.any() завершится с ошибкой




-------------------- Promise.race() --------------------

Используют, чтобы запустить несколько промисов 
    и получить результат первого выполненного
    в независимости от того, завершился промис с ошибкой или нет

Если первый завершился с успехом, то обработается в then()

Если первый завершился с ошибкой, то обработается в catch()

Если передать пустой массив, то промис навсегда окажется в состоянии pending
