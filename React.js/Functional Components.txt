Жизненный цикл компонента

В функциональных компонентах управлять жизненным циклом можно через хук useEffect

Жизненный цикл состоит из нескольких этапов:

    1. Монтирование (componentDidMount)
        выполняется с пустым массивом зависимостей( [] )
        выполняется когда компонент вмонтировался

    2. Обновление (componentDidUpdate)
        зависимости в массиве, и исходя из зависимостей компонент перерендеривается
            обновится

        Интересный момент: функция все равно вызовется в начале,
            даже если массив не менялся

    3. Размонтирование (componentWillUnmount)
        чтобы выполнить что то в конце жизни компонента нужно вернуть функцию с тем,
            что мы хотим сделать в конце
        return () => {}
