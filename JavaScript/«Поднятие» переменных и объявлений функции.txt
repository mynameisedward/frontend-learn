«Поднятие» в JavaScript — это когда мы запускаем код и под капотом компилятор поднимает 
    объявление переменных и функций наверх  
Поднятие в JS - возможность получать доступ к функциям и переменным до того, как они были созданы. 
Это механизм относится только к объявлению функций и переменных.

например 
// letsGo()
// .....
// function letsGo() {
//     console.log('letsGO')      
// }

Код выполнится!
То есть, "под капотом" компилятор JS "поднимает" все строчки, где объявляются функции на самый верх.

Применительно к функциям, "поднятие" работает только с объявлением функций!
Поднятие в JS не работает при использовании функциональных выражений, стрелочных функций и любых других способов создания функций...
Т.е работает только с function addNumber() {}


С переменными «поднятие» работает так: 
только с var

например мы хотим получить доступ к переменной внизу если это var — будет undefined
    т.к переменная есть, но значение мы получить пока не можем
