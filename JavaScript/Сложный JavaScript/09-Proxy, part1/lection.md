Proxy - класс, который позволяет создавать ловушки для объектов, функций, классов и т.д


В конструктор класса Proxy передается 2 параметра:
    1) target, цель к которому применяется Proxy
    2) Набор handle'ров, методы которые позволяют сделать ловушки для объекта


В функциях есть метод apply(), который вызывается по завершению функции
    Принимает: 
            target - тело функции
            thisArg - не знаю
            args - аргументы, котоые передаем
