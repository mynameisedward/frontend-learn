localStorage - локальное хранилище
    крутость в том, что там можно хранить любые данные,
        которые будут доступны после перезагрузки страницы


В node.js нет localStorage, это только возможность браузера

В DevTools можно посмотреть, что хранится в localStorage

localStorage работает для текущего домена


Взаимодействуем через объект localStorage:

- getItem('key')
    Получить значение элемента

- setItem('key', value)
    Добавить значение в localStorage

- removeItem('key')
    Удалить поле по ключу




localStorage работает только со значениями типа String
             работает только со строками

чтобы положить объект в localStorage, нужно преобразовать его в строку через JSON.stringify()
