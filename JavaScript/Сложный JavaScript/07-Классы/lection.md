ES6-классы

В классах мы можем реализовать методы, которые попадут в прототип экземпляра    

Также в классах можно указывать static переменные, которые можно получить только из самого класса, а не из экземпляра


--------------- Наследование классов -------------------

class Cat extends Animal

экземпляр класса Cat имеет все свойства и методы класса Animal
