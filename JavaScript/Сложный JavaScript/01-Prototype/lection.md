                  Object.create, наследование, цепочка прототипов 

У любого объекта есть свойство __proto__, которое является ссылкой на прототип объекта

Прототип - каждый объект имеет ссылку на другой объект - прототип, с помощью которого мы расширяем      свойства объектов или классов

Мы можем добавлять новые свойства в Object.prototype



Любые вещи которые создаются в javascript являются объектами
    Например, когда создаем строку, в ней уже в __proto__ присутствуют свойства и методы


Можно создавать цепочку прототипов
    Но есть 2 условия:
        1. Ссылки не могут идти по кругу, а то будет ошибка
        2. Значение __proto__ должно быть либо объектом либо null
