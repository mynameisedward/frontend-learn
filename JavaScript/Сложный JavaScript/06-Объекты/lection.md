Объект можно создать через Object.create()
    Метод принимает 2 параметра, которые являются объектами

В первом можем указать прототип
    Например, можем создать метод    

        
Во втором мы можем создать свойства и методы объекта 
    Свойства задаются по другому, пример:
        name: {
            value: 'Edward',
        },
        birthYear: {
            value: 2008
        }
    Здесь я задал ключи name и birthYear

Но эти ключи не итерируемые    
    Чтобы сделать их итерируемыми, мы должно задать свойство enumerable: true

Еще свойства:
    enumerable - чтобы свойсто было итерируемым
    writable - чтобы менять свойство
    configurable - чтобы удалять свойство
