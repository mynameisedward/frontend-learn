Генераторы - функции, которые мы можем приостановить и потом заново запустить
    Генераторы последовательно выдают результат своей работы
    Генераторы хранят своё состояние

Как результат работы генератора, мы получим объект next
    Который хранит 2 ключа:
        value - значение 
        done - true or false


В генераторе определен Symbol.iterator, который позволяет работать с циклом for .. of
