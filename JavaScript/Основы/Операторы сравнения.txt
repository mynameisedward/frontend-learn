Результат сравнения всегда имеет логический тип(boolean): true/false

Результат сравнения можно присвоить переменной


Как сравниваются строки? посимвольно

Берем первые буквы и сравниваем. 
Если первая буква первого слова больше первой буквы второго слова(выше в алфавите): 
    Значит первое слово больше второго
Если равны — двигаемся дальше по буквам

Замечание: Маленькая буква а("a") больше большой буквы A("A")


При сравнении разных типов — типы приводятся к числам
Пример:
    alert('2' > 1) // true, "2" становится 2

Также true становится 1, а false становится 0



Оператор строгого равенства === проверяет равенство без приведения типов.


Null и undefined
При строгом равенстве(===) они не равны. false

При нестрогом равенстве(==) они равны. true

При использовании математических операторов и других операторов сравнения < > <= >=
Значения null/undefined преобразуются к числам: null становится 0, а undefined – NaN.


Сравним null с нулём:

alert( null > 0 );  // (1) false
alert( null == 0 ); // (2) false
alert( null >= 0 ); // (3) true


Значение undefined несравнимо с другими значениями:

alert( undefined > 0 ); // false (1)
alert( undefined < 0 ); // false (2)
alert( undefined == 0 ); // false (3)
