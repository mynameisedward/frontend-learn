3 основных преобразования примитивов: строковое, численное и Логическое

Строковое - приводим к строковое


Численное - приводим к числу
    undefined -> null
    true / false -> 1 / 0
    '123' -> 123
    null -> 0


Логическое - приводит к true или false
    Все интуитивно пустые значение(undefined, null, '', 0)  ---> false
    Все остальное  ---> true
