=========== ОПЕРАТОР НУЛЕВОГО СЛИЯНИЯ ?? ===========

Значение определено если оно не равняется null/undefined

Результат a ?? b будет:
    если a определено — то а
    если а не определно — то b 
Грубо говоря, ?? возвращает первый операнд который не null/undefined


result = a ?? b
можно написать как 
result = (a !== null && a !== undefined) ? a : b;

Можно использовать больше двух

Как правило, ?? нужен для того, чтобы задать значение по умолчанию 
    для потенциально неопределенной переменной
Например:
    let user;
    alert(user ?? 'Anonim') // Anonim


Разница между || и ??
    || возвращает первое истинное значение
    ?? возвращает первое определенное значение(не null/undefined)



=========== ОПЕРАТОР НУЛЕВОГО ПРИСВАИВАНИЯ ??= ===========

x ??= y

x присвоится значение y если x не определено(null/undefined)



Оператор нулевого слияния ?? — это быстрый способ выбрать первое «определённое» значение из списка.
