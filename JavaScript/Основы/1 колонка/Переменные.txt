Для «var» не существует блочной области видимости

Область видимости переменных var ограничивается либо функцией, либо, 
    если переменная глобальная, то скриптом. Такие переменные доступны за пределами блока.


«var» обрабатываются в начале запуска функции
    т.е неважно, где объявлена переменная var, если объявлена хоть где то — она есть   
    с let такое не прокатит

    НО! Объявления переменных «всплывают», но присваивания значений – нет.
function sayHi() {
  alert(phrase);

  var phrase = "Привет";
}
sayHi();  // вот здесь будет выведено undefined


Для let существует блочная область видимости!
    Значит переменная видит только в пределах блока!


Const также использует блочную область видимости!
    но переменные объявленные с const нельзя изменять, 
        за исключением объектов и массивов
