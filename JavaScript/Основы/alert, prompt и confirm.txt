В JavaScript есть 3 модальных окна: alert, prompt и confirtm

Alert - оповещение
Просто выводит текст

Prompt - модальное окно с полем для ввода
Возвращает введеные пользователем данные, либо null если пользователь нажал Esc или отмена

Confirm - подтверждение
Возвращает true если пользователь нажал на ок, либо false если пользователь нажал на отмена или Esc


На мой взгляд это примитивные модальные окна, и я практические не замечаю такие 
    при повседневном использовании веб-сайтов
