
Для работы с концом массива: (выполняются быстро)
    pop - удаляет последний элемент и возвращает его
    push - добавляет элемент в конец (или несколько элементов)

Для работы с началом массива: (выполняются медленно)
    shift - удаляет первый элемент и возвращает его
    unshift - добавляет элемент в начало (или несколько элементов)



JavaScript хранит элементы массива в непрерывной области памяти 



length - это не количество элементов массива
length - это наибольший цифровой индекс плюс один
