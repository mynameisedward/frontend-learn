1. Метод map
    На основе одного массива делает другой
        Принимает функцию, которая описывает как преобразовать каждый элемент массива


2. Поиск элементов
    Есть 10 методов для поиска элемента

    Можно искать по значению, или по какому-то свойству

    -------- Поиск по значению --------

    1. includes - проверяет, есть ли элемент в массиве
    2. indexOf - находит индекс первого элемента
    3. lastIndexOf - находит индекс последнего элемента


    -------- Поиск по некоторому свойству(предикату) ---------

    1. filter - находит все элементы, удовлетворяющие свойству
    
    2. find - находит первый элемент, удовлетворяющий свойству
    3. findLast - находит последний элемент, удовлетворяющий свойству

    4. findIndex - находит индекс первого элемента, удовлетворяющий свойству
    5. findLastIndex - находит индекс последнего элемента, удовлетворяющий свойству
    
    6. some - проверяет, что в массиве впринципе есть элемент, удовлетворяющий свойству
    7. every - проверяет, что в массиве все элементы удовлетворяют свойству




3. Получение элементов массива

    1. at - возвращает один элемент по индексу

    2. slice - возвращает кусочек [start, end(не включительно)]

    3. join - делает строку, соединяя элементы массива
        можно передать символ соединения



4. Добавление и удаление элементов

    1. push - добавить в конец
    2. pop - удалить из конца
    3. unsfhit - добавить в начало
    4. shift - удалить из начала


    5. splice - добавляет и удаляет элементы в произвольное место в массиве
        
        array.splice(start, deleteCount, item1, item2, ...)
        
        начиная с индекса `start`, 
            удалить `deleteCount` элементов, 
                и поставить на их место `item1`, `item2', ....

        Мутирует исходный массив
            и возвращает новый массив


    6. fill - заполняет массив целиком или частично каким-то значением

        array.fill(value, start, end)

            value - каким значением заполнить
            start(optional) - откуда начать(optional)
            end(optional) - где закончить, не включительно

        Мутирует исходный массив


    7. toSpliced - как splice, но возвращает новый массив, а не мутирует исходный

    8. with - меняет один элемент массива и возвращает новый массив

        const newArr = arr.with(2, 'x')

    9. concat - объединяет несколько массивов в один

        const arr = [1, 2, 3]
        const newArr = arr.concat([4, 5], 6) // [1, 2, 3, 4, 5, 6]

        возвращает совершенно новый массив

        но spread прикольней
        

5. Изменение порядка элементов в массиве

    1. toReversed - создает новый массив, с элементами в обратном порядке
    2. reverse - тоже самое, но мутирует исходный массив
    3. toSorted - создает новый массив, с отсортированными элементами
    4. sort - сортирует элементы в массиве напрямую


6. Универсальные перебирающие методы

    1. forEach - для каждого элемента вызывает функцию колбэк

        arr.forEach(callback)

        currentValue - текущий элемент
        index - индекс
        array - сам массив

        - forEach нельзя остановить командой break!


    2. reduce - перебирает элементы, чтобы в конце получить одно значение

        arr.reduce(callback, initialValue)
        
        если нет initialValue, то берется первый элемент массива
    
    3. reduceRight - тоже самое, но перебор идет справа налево


7. Уменьшение вложенности 

    1. flat - поднимает вложенные массивы в массиве наверх

        arr.flat(depth)

        arr.flat() - поднимает на один уровень
        arr.flat(Infinity) - полностью выпрямляет массив
