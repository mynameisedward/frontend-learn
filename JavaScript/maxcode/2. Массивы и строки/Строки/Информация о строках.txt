----------------- Методы строки ----------------- 

1. endsWith - Заканчивается с 
    (string) => boolean
    
    
2. includes - Включает в себя
    (string) => boolean
    

3. localCompare - Сравнивает две строки

    str.localCompare(target)

    (target) => (-1 | 0 | 1)

    Если str должна идти перед target    -1
    Если строки равны                     0
    Если str должна идти после target     1

    Возможно полезно для сортировки строк в лексикографическом порядке


4. padEnd - добавляет символы к концу строки, пока ее длина не достигнет targetLength
    (targetLength, padString) => string


5. padStart - тоже самое, только для начала


6. replace - заменить часть строки на другую строку
    
    1 вариант -- заменить первое вхождение 
        str.replace('яблоки', 'груши') - поменяет первое вхождение `яблоки` на `груши` в строке str

    
    2 вариант -- использование функции для замены
        str.replace('яблоки', (match) => '$' + match)   

    Метод возвращает новую строку


7. replaceAll - тоже самое, но меняет все вхождения


8. slice - взять кусочек

9. at - взять по индексу
    можно также отрицательный

10. charCodeAt -
    Взять ASCII индекс
