1. Compose - функция, которая принимает другие функции и строит из них цепочку с помощью reduceRight
  Потом можно передать аргумент, и аргумент попадет в эту цепочку

  Пример в compose.
  

2. Для декоратора-мемоизатора использовать JSON.stringify как ключ в Map, 
  чтобы работать и с объектами тоже



3. Каррирование - разбить функцию с n-аргументами на n функций


4. sum(1)(2)(3)


5. Чтобы сделать так, чтобы одна функция была методом другой функции, 
    нужно возврщать объект из исходной функции, с методами

    function func1(value) {
      return {
        func2: function {
          console.log('something')
        }
      }
    }


    и можно будет вызывать функции вот так:
      func1(args).func2()
