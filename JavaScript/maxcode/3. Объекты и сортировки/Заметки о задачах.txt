1. Если ключ - строка
    obj[key]


2. Оператор in
    говорит есть ли свойство в объекте
    'prop' in obj
    проверяет в прототипах тоже


3. Ключи в объекте это:
    1) Строка (String)
    2) Символ (Symbol)


4. Если колбэк в map это несколько строк, то
    arr.map((elem) => {
        something
        something
        something
    })

    Фигурные скобки!!


5. Object.entries делает из объекта --> массив массивов
    Object.fromEntries делает из массива массивов --> объект


6.  Вместо {
        first: first,
        second: second,
        third: third
    }

    можно написать {first, second, third}


7. Ваще важно
    Чтобы пройтись по паре ключ-значение у объекта
    for(const [key, value] of Object.entries(obj)) {

    }


8. С помощью for .. in можно пройтись только по ключам
    for(const key in obj) {

    }


9. Сделать из сета массив -- [...set]

    Просто через спред


10. Обращение через obj.prop и через obj[prop]

    всё, что находится внутри [] -- вычисляется

    когда имя ключа хранится в переменной -- использовать obj[prop]
    а так - obj.prop
