1. Строки можно делить по символам 
    Из строки в массив 
    "Johnathan Due".split(' ')    =>    ['Johnathan', 'Due']


2. Замена символов строки
    Делаем из строки массив -> меняем элементы -> обратно в строку
            split()            ------>                  join()

3. Slice
    Взять кусочек массива или строки
    start - включительно
    end(optional) - не включительно

    str.slice(-4) возьмет 4 с конца


4. str.repeat(x)
    Повторить строку x раз


5. Math.max 
    Использовать с массивом только через ...


6. Чтобы удалить элементы из массива можно использовать filter
    filter принимает функцию true/false
