Функция-конструктор — обычная функция, которая создает объект.
Есть правила:
    1. Имя функции должно начинаться с заглавной буквы 
    2. Вызывать функцию можно только с помощью оператора «new»

Например:
function User(name) {
    this.name = name
}
const user1 = new User("Edward")


Цель конструктора — написать код для реализации большого кол-ва объектов, похожих друг на друга


Обычно в конструкторах нет return. 
    Но всё же, если он есть и возвращает объект — то вернется этот объект, замещая this
    Если же возвращает примитивное значение — оно проигнорируется, возвращая this.
Обычно в конструкторах отсутствует return
