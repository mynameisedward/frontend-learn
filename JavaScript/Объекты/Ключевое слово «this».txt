Ключевое слово «this» нужно для того, чтобы использовать свойства и метода объекта


Объявить метод в объекте: 

let user = {
    sayHi: function(){ // объявление функции
        console.log('Hello there!')
    }
}

эквивалентно:

let user = {
    sayHi(){ // объявление функции
        console.log('Hello there!')
    }
}


Ключевое слово «this» можно использовать также просто в функциях

Также заметим, что стрелочные функции являются особенными – у них нет this. 
    Когда внутри стрелочной функции обращаются к this, то его значение берётся извне.
