NaN — это вычислительная ошибка
Результат неправильной математической операции
например console.log('не число' / 2) // Выведет NaN

Значение NaN прилипчиво, это значит, что любое выражение с NaN возвращает NaN


Тип BigInt нужен для работы с числами произвольной длины
чтобы создать значение типа BigInt нужно добавить n в конце числа, например:
const bigNumber =  1230897120387129837619287123n


Тип null Это просто специальное значение, которое представляет собой «ничего», «пусто» или «значение неизвестно».


Тип undefined означает, что «значение не было присвоено».

Обычно null используется для присвоения переменной «пустого» или «неизвестного» значения, 
    а undefined – для проверок, была ли переменная назначена.
