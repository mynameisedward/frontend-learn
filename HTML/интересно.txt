box-sizing: есть 2 значения
    content-box -- это когда в width и height принимаются контент + padding и border элемента
        то есть элемент не будет width: 200px 
        он будет 200px + padding + border

    border-box -- это если мы задали width и height - элемент четко будет равен этим значениям

    Многие разработчики предпочитают использовать border-box по умолчанию 
        для более предсказуемого поведения элементов на странице.
