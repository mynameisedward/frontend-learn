Заметки:
    - при объявлении интерфейса(interface) писать I, например IUser
                                а тип(type) через T, например TBase

    - можно расширить интерфейс через extends 

    - интерфейс не может выступать форматом одного типа данных, как type

    - typescript не будет ругаться если мы вернем что-либо из функции, когда тип void

    - типизировать function declaration можно только при объявлении
